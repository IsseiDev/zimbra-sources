Performing an Advanced Search

The Advanced Search is an extension of the quick search. Advanced search allows you to perform more complex searches using different panes that apply to the type of content you want to search. You can open multiple instances of each pane, for example, if you want to do AND type searches.

  1. In the Search text box, enter the content that you want to search for. For search options, see Searching with Key Words, Search Syntax, and Using * as a Wildcard in Search.

  2. Select which part of your account to search from the drop-down menu next to the Search text box.

  3. Click the magnifying glass. The advanced search dialog displays.

  4. Enter the search criteria in the pane's text fields.

    5. If you enabled Always Show Search String on the Preferences>General page, the search text box updates as you type or make selections.

    As you enter information in a search pane, the search executes and continues to update as you enter for information. The results are shown in the Content pane.

  5. Click Search.
