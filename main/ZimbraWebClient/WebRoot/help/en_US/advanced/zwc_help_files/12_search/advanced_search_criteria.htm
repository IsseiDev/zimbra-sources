Advanced Search Criteria

Refer to the table below for advanced search criteria.

 

Option

Description

Basic Filters

Select criteria such as having an attachment, is flagged, or is unread.

Attachment

Search for email based on whether it has an attachment. If you are searching for a particular type of attachment, a list of formats is displayed.

Date

Search for messages received after, on, or before a selected date. To search between dates, use two Date panes.

Domain

Search for certain domains in the message header (From, To, and Cc). The domains that you can search for are predefined.

 

Folder

Limit search to selected folders.

Saved Search

Open a search query that you previously saved.

Size

Search for messages by size (bytes, KB, or MB).

Zimlets

Search for messages or contacts containing URLs, phone numbers, or other objects as defined by your administrator.

Status

Search for messages or contacts based on their status, such as read, unread, forwarded, or replied to.

Tag

Search for messages or contacts that have a specific tag.

Time

Search for messages that arrived within a time frame, such as the last hour, this week or last month.
